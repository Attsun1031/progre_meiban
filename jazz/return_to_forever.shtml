Return to Forever (リターン・トゥ・フォーエヴァー)

◎国籍

アメリカ

◎概要

Return to Foreverはアメリカのジャズロックバンド。
エレクトリック・ジャズの草分け的なアルバムである、マイルス・デイヴィスの「Bitches Brew」でエレピを担当したチック・コリアとスタンリー・クラークやアイアート・モレイラといったメンバーにより創設されました。
他にも、レニー・ホワイトやアル・ディ・メオラ、スティーブ・ガッドといったジャズ界の名プレーヤーが多く在籍しました。

同じく、アメリカのジャズロックバンドとして知られる「 マハヴィシュヌ・オーケストラ 」よりはジャズ色の音色が強いです。

「ライト・アズ・ア・フェザー」、「浪漫の騎士」あたりはフュージョンを語る上で欠かせない名盤です。
